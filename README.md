# ♟ «Эндшпиль»

Пошаговый рогалик, где игрок — шахматная фигура, меняющая форму (пешка → конь → слон → ладья → ферзь → архиепископ → канцлер → изверг), спускающаяся по процедурным этажам с биомами, врагами, особыми клетками, статусами, лутом, челленджами и мета-прогрессией.

Проект собран под **Vite** с тестами на **Vitest**. Звуки — синтез через Web Audio API. Генерация — детерминированный Mersenne Twister.

---

## 🎭 Нарратив:

### Мир

Этой игры нет в книгах. Она идёт в **Подземелье под Полем Брани** — бесконечной яме, куда ссылают души полководцев, проигравших свои войны. Наверху, в мире живых, всё ещё гремят битвы, и каждый погибший солдат падает не в землю, а в это подземелье. Здесь его душа сжимается в фигуру. Кость к кости, плоть к плоти.

Партия идёт веками. Два Игрока — **Красный Король** и **Чёрный Король** — два демона, заключённые в тронных залах на противоположных концах Подземелья. Они не могут двигаться. Они могут только приказывать. Их фигуры — мёртвые полководцы, которым обещали покой за победу. Покоя нет. Даже когда партия заканчивается, она начинается заново. Поле переворачивается, фигуры возвращаются на исходные, и бойня продолжается.

> _Это не спорт. Это камера пыток бесконечным сражением._

### Ты

Ты — **Пешка**, не помнящая имени. Ты был генералом. Ты проиграл битву, умер и попал сюда. Первые сто лет ты маршировал вперёд и умирал. Потом что-то сломалось. Может быть, удар пришёлся по шву, скрепляющему душу. Ты очнулся во **Рве** — яме за пределами доски, куда сбрасывают снятые фигуры. И понял: ты можешь брать кости других и приращивать к себе. Ты можешь **менять форму**.

> _Ты больше не фигура. Ты — Перевёртыш._

Твоя цель: пройти через всё Подземелье, добраться до Тронного Зала и убить одного из Королей. Не для победы. Для того, чтобы **разорвать петлю**.

### Что такое Цугцванг здесь

**Цугцванг — это Голод.** Каждая фигура в Подземелье обязана двигаться, потому что если она остановится, её начинает пожирать Тьма под доской. Ты чувствуешь это как голодную пульсацию в костях. Стоишь на месте — конечности начинают рассыпаться. Двигаешься — живёшь. Поэтому каждый ход — это выбор между смертью от врага и смертью от голода. Шкала ходов — это шкала сытости.

### Три акта: Нисхождение

#### Акт I — Ров

Ты выбираешься из ямы со сброшенными фигурами. Вокруг — горы обломков: рёбра, черепа, пальцы, спрессованные в стены. Враги здесь — **Свежие Пешки**: души, умершие недавно, ещё помнящие свои имена. Они стонут, когда ты их разбиваешь. Некоторые молят: _«Добей»_. Это не бой, это добивание.

Здесь ты находишь **Первую Чужую Кость** — фрагмент Коня. Приращиваешь его к плечу — боль, хруст, и ты уже можешь скакать буквой «Г». Но кость помнит прежнего хозяина. Ты слышишь его голос в голове несколько ходов после превращения.

> **Босс акта — Слон-Мучитель.** Он был инквизитором при жизни, теперь — надзиратель Рва. Он сшит из трёх тел, ходит по диагоналям и говорит голосами своих жертв одновременно. Когда ты его разбиваешь, он рассыпается на три Пешки, которые пытаются убежать. Ты можешь добить их или отпустить.

#### Акт II — Жернова

Ты входишь в действующую зону Подземелья. Здесь идёт бесконечная партия. Коридоры — линии доски, но искажённые: вертикали из ржавых цепей, горизонтали из запёкшейся крови. Фигуры здесь не автономны — они дёргаются, как марионетки, потому что Игроки всё ещё дёргают их за нити.

Ты видишь, как Пешек бросают вперёд без приказа. Как Ладьи врезаются в стены, потому что Король не видит доску. **Это конвейер смерти.**

Здесь ты находишь **Спаянных Ладей** — два генерала, которые при жизни были братьями и предали друг друга. Теперь они срослись спинами, обречённые ходить только вместе. Они не нападают первыми. Они защищают проход и кричат друг на друга голосами, полными ненависти. Ты не можешь убить одного — второй тут же бьёт в ответ. Ты должен найти позицию, где они блокируют друг друга.

#### Акт III — Тронный Зал

Ты спускаешься ниже, чем кто-либо из фигур. Здесь нет доски. Только **Тронный Зал Красного Короля** — огромная пещера, где на троне из собственных костей сидит демон. Он не может встать — его ноги вмурованы в пол, потому что таково правило: _Король ходит на одну клетку_.

Он говорит с тобой. Он устал. Он не злой. Он такой же пленник, как и ты. Просто он — сердце петли. Пока он жив, партия не кончается.

Финальный бой — не мат. Ты должен пройти через его **Королевскую Свиту**: Ферзь (его бывшая жена, добровольно ставшая чудовищем ради него), Ладьи (его дети, слепые и безмолвные), Кони (его рыцари, сошедшие с ума от вечности). Каждая смерть — диалог. Каждый умоляет или проклинает.

Когда ты добираешься до Короля, он предлагает **выбор**:

| Путь                   | Последствия                                                                                                                                                                          |
| ---------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------ |
| **⚔ Убить его**        | Партия кончится навсегда. Все души освободятся, включая тебя. Подземелье рухнет.                                                                                                     |
| **♚ Занять его место** | Ты садишься на трон, приращиваешь его кости к себе. Открывается режим «Новый Король»: ты начинаешь забег Королём, который может приказывать врагам (механика подчинения).            |
| **💥 Сломать доску**   | Если собраны все Кости Павших — разбиваешь пол Тронного Зала. Подземелье оказывается черепом мёртвого бога, а Игроки — его снами. При следующем запуске — новые, невозможные фигуры. |

## Запуск

```bash
npm install
npm run dev      # локальный дев-сервер (Vite)
npm run build    # прод-сборка в docs/
npm test         # прогон всех тестов (Vitest)
npm run lint     # ESLint проверка
npm run format   # Prettier проверка
npm run deploy   # сборка + коммит + пуш (GitHub Pages)
```

## Особенности

- **Голод (Цугцванг)** — шкала сытости, тает каждый ход; взятия и Жилы насыщают; виньетка тьмы
- **Кампания из 18 ярусов** — 3 акта с сюжетом, боссами и тремя концовками; бесконечный режим
- **4 босса** — Слон-Мучитель (ярус 5), Спаянные Ладьи (8), Жернов (11), Красный Король со свитой (18)
- **Нарративная система** — script.js с репликами врагов, голосами костей, интерлюдиями и эпилогами
- **Speech-бабблы** — всплывающие реплики над фигурами с fade-in/out и очередью
- **18 модулей** — полное разделение логики, рендера, UI и аудио
- **11 играбельных форм** — 5 стандартных + 3 экзотические + 3 новые (король, лазутчик, бастион)
- **6 биомов** — Залы, Коридоры, Лабиринт, Решётка, Арена, Пилоны (6 стилей стен)
- **5 челлендж-режимов** — Одинокая фигура, Слепой спуск, Шторм, Хаотичное колесо, Эскалация
- **11 типов спец-клеток** — паутина, портал, Жила, лёд, лава, туман, конвейер, ворота, плита, цветозона, жернов
- **6 особых врагов** — страж, некромант, двойник, ассасин, жрец, морозный маг
- **Камера с viewport** — поддержка карт больше экрана с плавным скроллом
- **Анимации** — перемещение фигур, частицы смерти, вспышка взятия, fade-переходы, анимированные тайлы
- **Звуки** — 8 синтезированных звуков через OscillatorNode
- **Мета-прогрессия** — Пепел, апгрейды, бестиарий, достижения (21), кодекс
- **Сохранение настроек** — звук и анимации в localStorage
- **Тултипы** — на врагах и спец-клетках (стеком)
- **Экран загрузки** — с логотипом, лором и подсказками
- **Мобильная адаптация** — тач-управление, крупные кнопки
- **Инструменты отладки** — модальное окно читов по слову `debug`
- **RNG на Mersenne Twister** — детерминированная генерация уровней
