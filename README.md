# ♟ Chess Roguelike

Пошаговый рогалик, где игрок — шахматная фигура, меняющая форму (пешка → конь → слон → ладья → ферзь → архиепископ → канцлер → изверг), спускаясь по процедурным этажам с биомами, врагами, особыми клетками, статусами, лутом, челленджами и мета-прогрессией.

Проект собран под **Vite** с тестами на **Vitest**. Звуки — синтез через Web Audio API. Генерация — детерминированный Mersenne Twister.

## Запуск

```bash
npm install
npm run dev      # локальный дев-сервер (Vite)
npm run build    # прод-сборка в dist/
npm test         # прогон всех тестов (Vitest)
npm run lint     # ESLint проверка
npm run format   # Prettier проверка
```

## Особенности

- **17 модулей** — полное разделение логики, рендера, UI и аудио
- **8 играбельных форм** — 5 стандартных + 3 экзотические (архиепископ, канцлер, изверг)
- **6 биомов** — Залы, Коридоры, Лабиринт, Решётка, Арена, Пилоны (6 стилей стен)
- **5 челлендж-режимов** — Одинокая фигура, Слепой спуск, Шторм, Хаотичное колесо, Эскалация
- **10 типов спец-клеток** — паутина, портал, руна, лёд, лава, туман, конвейер, ворота, плита, цветозона
- **6 особых врагов** — страж, некромант, двойник, ассасин, жрец, морозный маг
- **Камера с viewport** — поддержка карт больше экрана с плавным скроллом
- **Анимации** — перемещение фигур, частицы смерти, вспышка взятия, fade-переходы, анимированные тайлы
- **Звуки** — 8 синтезированных звуков через OscillatorNode (ход, взятие, смерть, ловушка, портал, руна, промоушен, лут)
- **Мета-прогрессия** — осколки, апгрейды, бестиарий, достижения, кодекс
- **Сохранение настроек** — звук и анимации в localStorage
- **Тултипы** — при наведении на спец-клетки
- **Экран загрузки** — с лором и подсказками
- **Мобильная адаптация** — тач-управление, крупные кнопки

## Архитектура

| Модуль       | Ответственность                              |
| ------------ | -------------------------------------------- |
| `config.js`  | CFG, глифы, имена, биомы, статусы, настройки |
| `state.js`   | Состояние `S`, хелперы                       |
| `dom.js`     | Кеширование DOM                              |
| `util.js`    | Хелперы, Mersenne Twister RNG                |
| `moves.js`   | Генерация ходов, угрозы                      |
| `board.js`   | Генерация этажа, спавн врагов                |
| `combat.js`  | Ход игрока, деградация, промоушен            |
| `enemies.js` | AI врагов, челленджи                         |
| `status.js`  | Статусы (яд/оглушение/щит/ускорение)         |
| `render.js`  | Canvas-рендер, камера, анимации              |
| `ui.js`      | Модалки, настройки, челленджи                |
| `loot.js`    | Реликвии, проклятия                          |
| `events.js`  | Комнаты-события                              |
| `meta.js`    | Мета-прогрессия, бестиарий                   |
| `content.js` | Реликвии, проклятия, апгрейды, челленджи     |
| `audio.js`   | Web Audio синтез звуков                      |
| `main.js`    | Точка входа, ввод                            |

npm test
